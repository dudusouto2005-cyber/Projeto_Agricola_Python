# Projeto_Agricola_Python
O monitoramento foi realizado em uma área experimental de 42 m² cultivada com tomate da variedade San Marzano.

Durante os sete dias de avaliação, a umidade variou entre 28% e 35%, apresentando média de aproximadamente 31%.

Esses resultados indicam que o solo permaneceu dentro da faixa considerada adequada para o desenvolvimento inicial da cultura.

A manutenção dessa umidade favorece o crescimento radicular, melhora a absorção de nutrientes e reduz o estresse hídrico das plantas.

Esse tipo de análise permite que o produtor decida o momento correto para irrigar, aumentando a eficiência do uso da água e reduzindo desperdícios.
